Online survey info

Programming languages: PHP MYSQL

In order to participate in the survey, the user must first register. After registration, the user logs in to the site and then chooses one of the three areas offered. Depending on the areas offered, the user receives a reward related to the chosen area (eeach user can only register once and receive only one reward, also the user cannot fill out the same survey more than once).
The project also includes an admin panel. Admin has the ability to view the list of users as well as the answers given by users.
